# NaVILA_project
This repo contains the scripts to install the NaVILA (High level only) repo.
The installation order is
1. Install_habitat-sim_new.sh
2. conda activate navila-eval
3. Install_habitat-lab_new.sh
4. Install_navila_new.sh

Note: That this repo's HW requirements are approx 40GB VRAM which means it wont work on your standard Laptop's GPU.
This repo ran successfully on an A6000.
